## Sobre OpenStock

OpenStock es una herramienta open-source que surge como alternativa a plataformas de mercado financieras de pago. Permite consultar precios en tiempo real, configurar alertas personalizadas y explorar información detallada de empresas, todo bajo una filosofía abierta y accesible.

Este repositorio **no es el proyecto oficial de OpenStock**. Su objetivo es servir como un espacio de prueba y aprendizaje, donde se documenta el uso de la herramienta, su funcionamiento general y las posibilidades que ofrece desde un punto de vista educativo.

El contenido de este repositorio está orientado a la experimentación, el estudio y la práctica, como parte del proceso de aprendizaje personal sobre plataformas de análisis de mercados y proyectos open-source.
